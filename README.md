# example-repo

## Description

This repository contains beginner JavaScript and PowerShell scripts created as part of the HyperionDev Software Engineering bootcamp.

### Files included:

- **hello_world.js** — A simple JavaScript program that prints "Hello World!" and "Git is awesome!" to the console.
- **file_cd.ps1** — A PowerShell script that creates folders, navigates into one, creates sub-folders, and removes two of them.
- **ifExample.ps1** — A PowerShell script that demonstrates the use of `if` and `if-else` conditional statements to check for folder existence and create new folders accordingly.
